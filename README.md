# 1ピクセルクエスト

> 「形が、運命。」

3×3グリッドにピクセルを配置してキャラクターの「形」を決め、横スクロールステージを攻略するパズル×ローグライクゲーム。

## ゲーム概要

- **編集フェーズ**: 3×3グリッドにピクセルを配置して形を作る
- **進行フェーズ**: キャラクターが自動で前進し、ギミックが形を変形・破壊する
- **全ピクセル消滅でゲームオーバー**

ステージが進むほどピクセル数が増え、ギミックも複雑になる。

## ギミック

| 種類 | 効果 |
|------|------|
| 天井 | 上段のピクセルを削除 |
| 底 | 下段のピクセルを削除 |
| 狭路 | 左右列のピクセルを削除 |
| 90°回転 | グリッド全体を時計回りに回転 |
| 重力 | ピクセルが下に落下 |
| 左右反転 | グリッドを左右反転 |
| シャッフル | ピクセルをランダム再配置 |
| 前方の敵 | 中段から攻撃 |
| 上方の敵 | 上段から攻撃 |
| 下方の敵 | 下段から攻撃 |

## 開発

```bash
npm install
npm run dev
```

## ビルド

```bash
npm run build
```

`dist/` に出力される。
